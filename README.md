# Slonimsky Pattern Lab

A scale-pattern generator for trumpet players, built on the system behind Nicolas
Slonimsky's *Thesaurus of Scales and Melodic Patterns*: divide the octave (or several)
into equal principal tones, decorate each one — interpolation, infrapolation,
ultrapolation — and it becomes a line.

**Use it here:** https://blewistrumpet-lang.github.io/slonimsky-pattern-lab/

- Every pattern family enumerated and engraved on a staff, written pitch for B♭ trumpet
- Filter families by the chord each pattern outlines
- Play, loop, practice over a root drone, nudge the root by half-steps
- Fold long patterns into trumpet range without breaking their shape
- Download any pattern (or a whole family) as MIDI

One self-contained HTML file — no dependencies, works offline.

Patterns are generated from Slonimsky's underlying (non-copyrightable) system;
this project contains no content from the book itself. Free for personal use.
